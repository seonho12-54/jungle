교육용 실험실

미니 리액트 흐름 탐험실

빈 컨테이너에서 첫 마운트가 일어나는 순간부터, 입력 이벤트가 state, memo, effect, diff, patch를 거쳐 실제 DOM으로 반영되는 과정을 버튼으로 한 단계씩 따라갈 수 있습니다.

어떻게 보면 좋나요?

  1. 먼저 시나리오를 고릅니다.
  2. `이전 단계 / 다음 단계`로 이벤트를 한 칸씩 움직입니다.
  3. 현재 설명, 값 이동, 직전/직후 상태, DOM, 실제 소스 코드를 같이 봅니다.
